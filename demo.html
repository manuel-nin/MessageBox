How to use

The parameters are the following:

MessageBox(text, type, close_redirect, buttons, button1_description, button1_redirect, button2_description, button2_redirect)

Where:

  • text: The text you would like to show in the MessageBox.
  • type: Defines which icon show in the MessageBox. Options are: 'info', 'success' and 'error'. More options can be configured inside the file 'messagebox/css/styles.css'.
  • close_redirect: URL to open when the MessageBox is closed by clicking the [x] icon. If you would like to call a JavaScript function you can set the value to: 'javascript:alert(123);' for example.
  • buttons: Defines how many buttons you would like to show in the MessageBox. Valid options are '1', '2' or ''. If this option is empty, no button will be shown.
  • button1_description: Content to display on the button #1.
  • button1_redirect: URL to go when the user clicks the button #1.
  • button2_description: Content to display on the button #2.
  • button2_redirect: URL to go when the user clicks the button #2.
